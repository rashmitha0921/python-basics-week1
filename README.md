# Week 1: Python Basics - Personal Introduction Program

## Project Overview
This project is part of Week 1 of Python Basics tutorial. It demonstrates fundamental Python concepts including variables, input/output operations, and string formatting through a simple interactive program that collects user information and displays a personalized welcome message.

## Learning Objectives
- Understand basic Python syntax and structure
- Learn to use variables for data storage
- Practice input() function for user interaction
- Use print() and f-strings for output formatting
- Create and run a complete Python program

## Technical Requirements
- Python 3.6 or higher
- No external libraries required
- Command line interface for execution

## Setup Instructions

### 1. Install Python
Python is already installed on your system (version 3.10.7). If you need to install it:

**Windows:**
- Download from https://www.python.org/downloads/
- Run the installer
- Make sure to check "Add Python to PATH"

**Verification:**
```bash
python --version
```

### 2. Clone/Download Project
Navigate to the project directory:
```bash
cd c:\Users\91973\Desktop\python-basics-week1
```

### 3. Run the Program
Execute the program:
```bash
python personal_intro.py
```
